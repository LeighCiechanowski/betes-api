# Betes-Bot Chat Bot API

Betes Bot is a chatbot service that aims to help diabetics manage their condition. Betes Bot will accept a users glucose readings and provide informative insights into that data.

### Installing

Install packages

```
npm install
```

Fire up mongodb

```
mongod
```

Run it

```
node server
```
